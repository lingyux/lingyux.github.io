媒体库

为什么要建自己的媒体库

  1. 为什么不使用爱奇艺，腾讯视频等流媒体？为什么不使用盗版的在线平台？
    • 码率低。
    • 删减问题。
    • 限制使用人数。
    • 资源质量差(屎里淘金)。
    • 周末在家闲着没事？🤓👆折腾一下。
  2. 直接下载到网盘或者硬盘不就好了，为什么要多此一举？
    • 分享不便。
    • 没有元数据。
    • 无法同步观看记录。
    • 多用户无法做到观看记录分离。

CloudDrive2

CloudDrive2 是一个云存储管理平台，可以多个网盘服务挂载为本地磁盘，从而方便地访问和管理云文件。

  1. 多网盘支持：
    • CloudDrive2 支持多种网盘服务，包括阿里云盘， Google Drive、OneDrive、Dropbox 等国内外网盘服务商，用户可以将这些网盘挂载为本地磁盘，统一管理。
  2. 大容量影音库：
    • 通过将网盘挂载到设备上，用户可以创建大容量的影音库，方便存储和播放大量的影音文件。

Emby

  1. 媒体管理：
    • Emby可以集中管理电影、电视剧、音乐、照片等各种媒体资源。自动刮削元数据，下载封面图和描述信息。
  2. 跨平台支持：
    • Emby支持多平台，包括Windows、Mac、Linux、Android、iOS， TVOS等。
  3. 用户管理：
    • Emby可以创建多个用户账户，并为每个账户设置不同的访问权限，每个人都可以拥有自己的观看记录。
  4. 实时转码：
    • Emby支持实时转码，可以根据设备和网络状况自动调整视频质量，确保流畅播放。
  5. 插件扩展：
    • Emby拥有丰富的插件库，可以根据需要安装各种插件，扩展Emby的功能，如字幕下载、媒体同步等。
  6. 远程访问：
    • 通过Emby，您可以在任何有网络连接的地方访问媒体库。它支持内网穿透和远程访问，可以随时随地享受影音内容。

使用

  1. 安装CloudDrive2
  2. 安装Emby
  3. 配置CloudDrive2与Emby：
    • 在CloudDrive2中，将网盘挂载为本地磁盘（如F:、G:等）。
    • 打开Emby服务器的管理界面，选择“媒体库”选项。
    • 点击“添加媒体库”，选择媒体类型（如电影、电视剧等），然后选择挂载的本地磁盘路径。
  4. 媒体刮削和管理：
    • Emby会自动扫描并刮削媒体文件，下载相关的元数据和封面图。
    • 在Emby的管理界面中对媒体库进行分类和管理。
    • 手动修改刮削错误的媒体信息
  5. 播放和访问：
    • 在任何支持Emby客户端的设备上（如智能电视、手机、平板等）安装Emby客户端应用。
    • 登录Emby账户，访问并播放个人影音库。

播放器

播放器的选择很多，IOS的SenPlayer，Infuse，Fileball等，Android的AfuseKt，Hills等，但高级功能都需要付费，除Infuse以外，终身版的售价普遍在100元左右。Emby也有自己的官方移动app，但需要付费而且体验不好。如果不添加别人搭建的大型emby服务器的话，个人使用一般不用担心版权问题，所以推荐网易出品的播放器。

网易爆米花是一款由网易开发的智能媒体库和全能播放器。

  1. 智能媒体库：
    • 支持聚合电影、电视剧、综艺等多种影视资源，自动识别影片信息，匹配海报、评分和简介。
    • 可以将阿里云盘、百度网盘、天翼云盘等多个网盘的内容聚合到一个媒体库中，方便管理。
  2. 全能播放器：
    • 支持超过200种音视频格式和字幕编码，无需下载即可流畅播放4K、HDR等超清视频。
    • 支持跨平台同步，用户可以在手机、电视、平板、电脑等多终端无缝观看。
  3. 无广告体验：
    • 网易爆米花致力于提供纯粹的视听体验，没有广告打扰，用户可以专注于观影。
  4. 多协议支持：
    • 支持WebDAV、SMB协议，方便用户导入本地目录和网盘中的影片。
  5. 支持全平台。

Ani

简介

Ani 是一款开源的跨平台弹幕追番应用，专为动漫爱好者设计。以下是它的一些主要特点：

  1. 多平台支持：Ani 支持 Android 和桌面系统（Windows 和 macOS）。
  2. 多数据源整合：支持多个视频数据源和弹幕服务，包括 BT 和在线资源，自动整合资源，提供高质量和快速缓冲的观看体验。
  3. 云同步：支持 Bangumi 收藏同步，记录观看进度，方便用户随时随地继续观看。
  4. 弹幕功能：内置弹幕引擎，支持自定义弹幕样式和速度，并提供公益弹幕服务器。
  5. 离线缓存：根据用户的资源偏好设置自动缓存下一集，方便离线观看。
  6. 开源免费
